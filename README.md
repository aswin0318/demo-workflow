# Sample Java Codebase ---

This repository contains a minimal Java application built with Maven.

## Structure

- `pom.xml` - Maven build definition
- `src/main/java/com/example/demo/App.java` - application entry point
- `src/test/java/com/example/demo/AppTest.java` - JUnit test

## Build and run

```bash
mvn clean verify
java -jar target/sample-java-app-1.0.0.jar
```
